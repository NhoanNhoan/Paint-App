# Paint-App
  Draw shapes(rectangle, square, circle, hexagon, ...)
  
# Prerequisite:  
1. Visual Studio(my version 2019).  
2. Dotnet C#: 4.6(my version).  
3. Good laptop(or computer) can run server and client.
